# MCDA5550 PWA Assignment

This project is a Progressive Web Application (PWA) built with **React** and **Vite**.  
It allows users to manage tasks with offline support using **IndexedDB**.

---

## Prerequisites

Before running the application, make sure the following software is installed on your system:

- **Node.js** (version 18 or higher recommended)
- **npm** (included with Node.js)

Verify installation using the terminal:

```bash
node -v
npm -v
```

## Install Dependencies

Open a terminal.

Navigate to the project root directory:

cd MCDA5550-PWA-Assignment/pwa

Install all required dependencies:

```bash
npm install
```

This command installs all libraries defined in package.json.

## Run the code

Input command below in terminal to run the PWA program.

```bash
npm run build
npm run preview
```
